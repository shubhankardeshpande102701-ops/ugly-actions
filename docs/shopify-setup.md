# Shopify Setup

Shopify is planned but not active in v1.

## Later Steps

1. Create a Shopify store for Ugly Actions.
2. Add the three starter products.
3. Add sizes and colors as product variants.
4. Configure shipping, taxes, returns, and store policies.
5. Create Storefront API credentials.
6. Add environment variables locally.
7. Replace local product data with Shopify product queries.
8. Add cart and checkout handoff.

## Why Not Now

The current website is a coming soon product preview. Waiting keeps the first build simpler and lets the design mature before real selling begins.
