# Project Brief

Ugly Actions is a premium unisex athletic wear brand. The first website is a local-first product preview for Drop 001.

## V1 Goal

Create a polished website that feels like a real premium storefront while avoiding live checkout until the brand, products, and Shopify setup are ready.

## V1 Products

- Action Training Short
- Ugly Rep Performance Tee
- After Action Training Hoodie

## Success Criteria

- The home page immediately communicates the Ugly Actions brand.
- Each product has a dedicated detail page.
- Visitors can join a drop list.
- Product pages show fit, size options, material direction, and future Shopify-ready product details.
- The project is easy for a beginner to understand and run locally.
