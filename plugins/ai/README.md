# AI Plugins

This folder tracks AI-related tools we use or plan to use with the Ugly Actions Next.js app.

The goal is to keep AI tooling organized instead of scattering notes, setup commands, and generated files across the app.

## Current Plugin

- `impeccable/`: design-quality skill and CLI for AI coding workflows.

## Planned Next.js AI Plugins

- `vercel-ai-sdk/`: future streaming AI routes and React AI UI helpers.
- `openai/`: future OpenAI API integration notes and environment setup.
- `browser-ai/`: future browser-side AI experiments such as Transformers.js.
- `rag-search/`: future product/brand knowledge retrieval notes.

Only Impeccable is installed right now. The others are placeholders so we can add them intentionally later.
